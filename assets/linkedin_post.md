# LinkedIn Post Draft

I built `how-llm-works`, a visual and interactive repository that explains how prompting and GPT-style language models work from first principles.

To make the learning concrete, I used banking fundamentals as the teaching domain:

- savings accounts
- loans
- interest rates
- risk management
- central banking

The repo includes:

- a mini GPT-style model in PyTorch
- end-to-end training and inference
- a Streamlit app for generation and attention visualization
- beginner-friendly docs and notebooks

The goal was simple: make LLM internals understandable without losing architectural correctness.

If you want to learn how tokenization, embeddings, self-attention, transformer blocks, and autoregressive decoding fit together, this project gives you a clean starting point.
